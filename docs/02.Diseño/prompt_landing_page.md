# Prompt para generar la Landing Page en Stitch / v0 / AI

Puedes copiar y pegar el siguiente texto directamente en la herramienta de IA (como Stitch) para generar el código de tu Landing Page:

---

**Actúa como un desarrollador frontend experto en React y Tailwind CSS, y como un diseñador UI/UX especializado en interfaces modernas y elegantes.**

**Objetivo:**
Crea una Landing Page para un negocio (Estética y Barbería). Esta página será el punto de entrada de un sistema web completo (ScheduleManager), por lo que el diseño y los estilos deben estar pensados para ser escalables y reutilizables en la aplicación interna (dashboard, formularios, tablas).

**Estilo Visual y Diseño:**
1. **Moderno y elegante:** Usa un diseño limpio, con espacios en blanco adecuados (whitespace) y tipografía legible y contemporánea.
2. **Paleta de colores (Familiaridad, cercanía y formalidad):** 
   - Utiliza colores que evoquen confianza y calidez (por ejemplo, tonos arena, beige cálido o madera clara).
   - Combínalos con colores que aporten formalidad y elegancia (como gris carbón oscuro, azul noche o un verde oliva profundo).
   - Mantén un fondo claro u *off-white* para la lectura cómoda.
3. **Componentización:** Escribe el código utilizando componentes funcionales de React para que los estilos (botones, tarjetas, tipografías) puedan reutilizarse fácilmente más adelante.

**Secciones Requeridas en la Landing Page:**
1. **Header / Navegación:** Logo o nombre del lugar a la izquierda. Botones de "Iniciar Sesión" y "Registrarse" a la derecha.
2. **Hero Section:** 
   - Un título atractivo (H1) que destaque.
   - Un subtítulo que explique brevemente el valor del servicio.
   - Un **Call to Action (CTA) principal** muy visible que diga: "Agendar una cita".
3. **Descripción del Lugar (Sobre Nosotros):**
   - Una sección que describa el ambiente del lugar (estética/barbería), destacando la atención personalizada, la experiencia y la cercanía con el cliente.
4. **Servicios Ofrecidos:**
   - Una cuadrícula (grid) o tarjetas (cards) resumiendo los principales servicios (ej. Cortes de Cabello, Arreglo de Barba, Tratamientos Faciales, etc.). Si es posible, incluye íconos descriptivos para cada uno.
5. **Ubicación y Contacto:**
   - Información de la dirección física del lugar.
   - Horarios de atención.
   - Un contenedor simulando un mapa (placeholder).
6. **Footer:**
   - Enlaces rápidos, redes sociales y derechos de autor.

**Requisitos Técnicos:**
- Utiliza **React** con **Tailwind CSS**.
- Asegúrate de que el diseño sea totalmente responsivo (Mobile-first).
- Agrega interactividad sutil en los hovers (ej. transiciones suaves en los botones y enlaces).
